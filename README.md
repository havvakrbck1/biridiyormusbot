A Telegram bot that shares random interesting facts with users.

- User-friendly interface
- Regular updates

## Technologies Used
- Python 3
- Telegram Bot API

## Installation

```bash
git clone https://github.com/havvakrbck1/did-you-know-bot.git
cd did-you-know-bot
